### How to build

#### Setup

- clone this repo

- install node 20

- install npm 10.8.2

- install the dependencies
```
npm install
```

- run project in local enviroment to play around
```
npm run dev
```

#### Build

```
npm run build
```

#### Tests

```
npm run test
```
